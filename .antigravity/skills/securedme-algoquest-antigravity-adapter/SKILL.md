# AlgoQuest Antigravity Adapter

Generated by SecuredMe Education adapter scaffold.

Use this skill for the `.antigravity` VS Code extension-like adapter in `algoquest-ams-discovry-labs-module-`.

## Workflow

- Canonical domain: `https://algoquest.securedme.ca`.
- Status: `pre-alpha` until app-specific validation is complete.
- Prefer browser-authenticated Google/Gemini/Antigravity workflow.
- Use native login only as fallback after WebAuth cannot complete.
- Use `petit-yolo-instructions` only as the final fallback handoff pattern.
- Never require student API keys.
- Never store token, cookie, browser session, API key, OAuth material, or `.env` values.

## Future Routes

- Token Governor route: `.antigravity/securedme-adapter-map.json`.
- WebAuth template: `.antigravity/webauth-template.json`.
- MCP route: planned only; this scaffold connects no live MCP server.
- Do not rewrite the official school mechanism from this app adapter.
